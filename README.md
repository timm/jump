# Jump

Non-parametric optimization.<br>
Find interesting bits. Combine them. Repeat.<br>

## Install

- Get Julia (> 1.5)
- Load [requirements.jl](requirements.jl)
- Change directories to the [src](/src) directory.
- Start julia
- `include("nb.jl")`
- Run the tests with `ok();`
